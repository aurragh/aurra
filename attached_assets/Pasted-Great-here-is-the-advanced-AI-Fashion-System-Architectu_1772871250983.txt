# AI Fashion Stylist Platform — System Architecture Prompt

Act as a **principal AI architect, machine learning engineer, full-stack engineer, and product designer**.

Design and implement a **scalable AI-powered fashion styling platform** that generates outfits, explains styling logic, visualizes outfits on avatars, and allows users to shop for similar items.

The system should combine:

• LLM reasoning
• computer vision
• vector search
• product databases
• 3D avatar rendering

The platform must function as an **AI stylist + virtual try-on + fashion discovery system**.

---

# 1. Core Product Flow

The platform workflow should follow this pipeline.

### Step 1 — User Style Quiz

User answers a quiz about:

• body type
• height
• skin tone
• style preference
• lifestyle
• climate
• fashion inspiration
• accessories preference

Output:

**Structured Style Profile**

Example:

```
{
 body_type: "pear",
 palette: "warm autumn",
 style: "minimal elegant",
 climate: "temperate",
 lifestyle: "corporate",
 accessories: ["watches","necklaces"]
}
```

---

### Step 2 — AI Outfit Generation

The system generates outfits using:

**LLM + style rules engine**

Inputs:

• user style profile
• fashion knowledge base
• seasonal rules
• body proportion rules

Output:

```
Outfit:
Top: Ivory silk blouse
Bottom: Navy high-waisted skirt
Shoes: Nude pumps
Accessories: Gold watch + minimal necklace
```

---

### Step 3 — Explanation Engine

Use the LLM to generate:

**“Why this works” explanation**

Example:

"This outfit balances your pear body shape by drawing attention upward through the blouse while the high-waisted skirt elongates the silhouette."

The explanation should consider:

• color theory
• body balance
• proportion
• occasion fit

---

# 2. Avatar Rendering System

The system should support **2D and 3D avatar visualization**.

---

# 5. Clothing Layering System

Layer order:

```
underwear
base clothing

```

# 6. Fashion Product Search System

The shop feature should use **AI visual search**.

Pipeline:

1. Extract features from outfit item
2. Convert to embedding vector
3. Query product database
4. Return similar items

Possible tools:

• CLIP embeddings
• OpenAI embeddings
• Pinecone / Weaviate vector database

Closest matching alternatives.

💡 **Important advice (this will save you months):**

Most startups building AI stylists fail because they start with **3D avatars first**.

Correct order is:

1️⃣ Style quiz
2️⃣ Outfit generation
3️⃣ Product search
4️⃣ Avatar visualization

Not the other way around.
