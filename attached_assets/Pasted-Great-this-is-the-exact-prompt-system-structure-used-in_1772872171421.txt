# The Professional AI Stylist Prompt System

Instead of **one prompt**, you create **five prompt layers**.

Each layer feeds the next.

```
User Data → Style Profile → Outfit Logic → Outfit Generation → Visualization
```

---

# Layer 1 — User Style Profile Prompt

This converts quiz answers into a **structured fashion identity**.

### Prompt Template

```
You are a professional fashion stylist.

Analyze the following user profile and create a structured style identity.

User data:
Height:
Body shape:
Skin tone:
Hair color:
Lifestyle:
Climate:
Fashion preference:
Budget range:
Accessories preference:

Return:

1. Style archetype
2. Best color palette
3. Ideal silhouettes
4. Recommended fabrics
5. Accessories that complement the user
6. Fashion mistakes to avoid
```

---

### Example Output

```
Style archetype: Minimal elegant professional

Best palette:
Cream
Navy
Soft gold
Muted olive

Silhouettes:
High waist
Structured jackets
A-line skirts

Avoid:
Oversized tops
Low waist pants
Heavy prints
```

This becomes the **foundation of the stylist AI**.

---

# Layer 2 — Outfit Reasoning Prompt

This is where the AI actually **thinks like a stylist**.

### Prompt Template

```
You are an expert fashion stylist.

Based on the user style profile below, generate 3 outfits.

Consider:

• body proportion balance
• color harmony
• occasion suitability
• layering logic
• accessories compatibility

User style profile:
[paste profile]

Return structured outfits.
```

---

### Example Output

```
Outfit 1
Top: ivory silk blouse
Bottom: navy high-waist midi skirt
Shoes: nude pumps
Accessories: gold watch

Outfit 2
Top: fitted olive blazer
Bottom: cream trousers
Shoes: brown loafers
Accessories: leather belt
```

---

# Layer 3 — Styling Explanation Prompt

This creates the **trust layer** of your product.

Example prompt:

```
Explain why the following outfit works for the user.

User body shape:
User color palette:
User lifestyle:

Outfit:
[top]
[bottom]
[shoes]

Explain using fashion logic.
```

Example output:

```
The high-waisted skirt elongates the lower body while the light blouse draws attention upward. The navy and ivory palette complements warm undertones and creates a balanced professional look.
```

Users trust AI **when it explains decisions**.

---

# Layer 4 — Outfit Visualization Prompt

This is used for **image generation / avatar rendering**.

Example prompt structure:

```
Full body fashion model wearing:

Top: ivory silk blouse
Bottom: navy midi skirt
Shoes: nude pumps
Accessories: gold watch

Style: editorial fashion photography
Background: neutral studio
Lighting: soft fashion lighting
Pose: standing model
Camera: full-body shot
```

Adding **camera + lighting + pose** dramatically improves results. ([Medium][2])

---

# Layer 5 — Product Matching Prompt

This connects the outfit to **shoppable items**.

Example prompt:

```
Find real fashion products similar to the following outfit.

Top: ivory silk blouse
Bottom: navy midi skirt
Shoes: nude pumps

Return:

• exact matches
• visually similar items
• affordable alternatives
```

This powers the **shop feature**.

---

# Advanced Prompt Technique (Used by Startups)

The best AI stylists use **prompt chaining**.

Example:

```
Step 1
Generate user style identity

Step 2
Generate outfits

Step 3
Explain outfit logic

Step 4
Generate avatar image prompt

Step 5
Find similar products
```

Each output feeds the next step.
