# 2. The 4 Core Product Engines

The system should be divided into **4 intelligent engines**.

---

# Engine 1 — AI Stylist Brain

This is the **core intelligence** of the platform.

Inputs:

* user quiz
* uploaded images
* body shape
* color palette
* style preference

Output:

* outfit recommendation
* styling explanation
* accessories suggestions

### Tech

LLM + rules engine.

Example prompt logic:

```

Warm autumn palette
Minimal corporate style

Explain why each works.
Suggest accessories.
```

This creates the **visual experience**.

Two modes:

Fast and simple.

### Mode 2

3D avatar

User can:

* rotate model
* swap clothing
* inspect items

Future feature:

Upload photo → create personal avatar.

Tools:

* Three.js
* ReadyPlayerMe
* MetaHuman
* SMPL body models

# Engine 3 — Fashion Search Engine

This connects outfits to **real products**.

Example:

AI generates outfit:

White blouse
Black skirt
Gold watch

System searches:

fashion retailers

Uses **vector similarity search** to find visually similar items.

Tech stack:

Vision model → embeddings → vector database.
